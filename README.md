# DSA with JavaScript

A collection of data structures and algorithms implementations in JavaScript, intended for learning and practice.

## Overview

This repository contains small, focused JS files implementing common algorithms and coding-challenge solutions (sorting algorithms, search algorithms, and some LeetCode practice solutions).

## Contents

- `binary.js` — Binary search implementation
- `linear.js` — Linear search / basic search example
- `bubble.js` — Bubble sort implementation
- `selection.js` — Selection sort implementation
- `insertion.js` — Insertion sort implementation
- `merge.js` — Merge sort implementation
- `twoSum.js` — Two-sum problem solution (common interview/LeetCode problem)
- `leetcode 9.js` — Solution for LeetCode problem 9 (filename contains a space)
- `sample.js` — Small sample / helper code


## How to run

Make sure you have Node.js installed (v12+). From the repository root you can run individual files with:

```bash
node filename.js
```

For example:

```bash
node merge.js
```
